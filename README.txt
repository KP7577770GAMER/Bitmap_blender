# Bitmap_blender

The program gets two bitmaps as input : the input method can be one of these

1. select two Bitmap (.bmp) files and drag onto the BmpBlender.exe and new file named merge.bmp will be created

2. command line input :- BmpBlend image_1.bmp image_2.bmp 0.5 image_merge.bmp
    (where the first argument is first image, second argument is second image, third is the ratio of blending and fourth is name for output file)
